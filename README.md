<h1 align="center">🚀 Ads Bot</h1>

<p align="center">
A powerful multi-session Telegram Ads Broadcasting Bot built with <b>Pyrogram</b> + <b>Telethon</b>.
</p>

<p align="center">
  🔐 Multi-User Hosting • 📢 Smart Broadcasting • 🐳 Docker Ready • 🧠 Profile Automation
</p>

---

## ✨ Features

- 👤 Multi User Hosting System  
- 🔐 Multiple Client Sessions Per User  
- 📢 Group & Channel Broadcasting  
- ⏱ Custom Delay System  
- 📊 Live Logs & Analytics  
- 📝 Auto Profile Update On Broadcast Start  
- 🔄 Per-User Isolated Broadcast Engine  
- 🎛 Clean Dashboard UI  
- 🐳 Docker Supported  

---

## 🛠 Tech Stack

- 🐍 Pyrogram
- ⚡ Telethon
- 🍃 MongoDB (Motor)
- 🔁 AsyncIO
- 🐳 Docker

---

## 📦 Installation (Local Setup)

```bash
git clone https://github.com/YOUR-USERNAME/Ad-Bot
cd Ad-Bot
pip install -r requirements.txt
python3 -m Nexa
```

---

## 🐳 Run With Docker

- Build image
```bash
docker build -t nexa-bot
```

- Run Container
```bash
docker run -d --name nexa-bot nexa-bot
```

---

## ⚙️ Environment Variables

- Create a .env file in root directory:
```bash
API_ID=your_api_id
API_HASH=your_api_hash
BOT_TOKEN=your_bot_token
MONGO_URI=your_mongo_uri
SESSION_DIR=sessions
```
